# Timber SFML Game

Классическая аркада в стиле **Timberborn/Timberman**, разрабатываемая на языке **C++** с использованием графической библиотеки **SFML**.

## О проекте
Цель игры — срубить как можно больше секций дерева, избегая столкновения с ветками. Проект создан для отработки навыков работы с игровым циклом, обработкой ввода, анимацией спрайтов и системой подсчета очков.

## Инструменты
*   **Язык:** C++
*   **Библиотека:** SFML (Simple and Fast Multimedia Library)
*   **Сборка:** Visual Studio
